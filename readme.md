# Tiny Heureka

## Použité technologie

- node
- pnpm
- Typescript
- Vite
- React
- React-Router
- React-Bootstrap
- Axios

## Spuštení:

`pnpm install`

dev server:

`pnpm run dev`

build

`pnpm run build`

## Co (ideálně) dodělat

- handlování chyb z API
- lepší načtení informací pro produkt (description, image, price)
- styly
- ...
